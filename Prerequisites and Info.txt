You need to install the packages/modules:
Numpy, matplotlib and pandas.

Anaconda install code run:

Pandas:
conda install -c anaconda pandas 

Numpy:
conda install -c anaconda numpy

Matplotlib:
conda install -c conda-forge matplotlib



In this example I compare the results of a Linear Regression Model VS a Polynomial Regression Model. Results are shown in numbers and in a visual-representation colored graph.
